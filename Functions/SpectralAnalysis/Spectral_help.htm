General  information

The top panel shows the name of the loaded data set and its dimensionality

The functions in this program only operate on columns.

The sample rate can be adjusted in the top panel

If you hover above some text fields tooltips are shown (e.g. how the select collumns)

 

 

 

Compare selected EEG signals in one axis

 

This function allows you to compare EEG in the same axis. This is for example useful to determine the magnitude of a difference.

Two plots are generated:

1.   The selected EEG signals are plotted as a function of time

2.   The selected EEG signals are visualized in the frequency domain

 

Use zoom and Data cursor to measure details….

 

There is more power in the 50 Hz range in the green signal than in the blue signal.

 

Colors: The color code that is used for lines is printed in the title of the top graph.

Column 1 is blue, column 2 is green etc.

 

Max 8 signals can be plotted

 

 

Plot selected EEG signals in the time and the frequency domain

This function plots EEG in the time and the frequency domain.

Two plots are generated.

1.   EEG in the time domain

2.   EEG in the frequency domain

 

All channels are plotted in individual linked x-axis.

Use zoom to focus on a time or frequency range of interest.

 

 

Generate a spectrogram of selected EEG signals

 

 

This function ads some time resolution to the FFT

It generates a three dimensional graphs with time on the x axis, frequency on the y-axis and power on the z-axis.

Power is visualized in color.

X-and Y-axis are linked in the plots

 

Input data fields:

·        Channel(s): Select the channels you want to plot (Tooltip)

·        Window: is a Hamming window of length nfft, max 256

·        Overlap: The number of samples that each segment overlaps. More overlap means a better time resolution but a longer calculations.

·         nfft: is the FFT length and is the maximum of 256 or the next power of 2 greater than the length of each EEG segment. nfft is number of sampling points to calculate the discrete Fourier transform. nfft must be a scalar.

·        2D filter size: the spectrogram is somewhat smoothened using a 2D Gaussian filter. Values of pixels around each pixel are used to determine the value of each filtered pixel. Only uneven values are allowed. Example: if 3, than one pixel at each side of the pixel of interest is used for the smoothening, if 5, two pixels at each side are used, if 1 no filter is applied.

 

Two plots are generated:

1.   The minimal and maximal power of frequency are mapped optimally on the color map. Frequency resolution is thus visualized in the best possible way.

2.   The same color map is used for all subplots. Minimal power of a frequency in any subplot determines a minimum, maximum power of frequency in any subplot determines a maximum. This minimum and maximum are mapped to a colormap.

 

 

 

Plot 1 is best to study EEG in individual channels.

Plot 2 allows you to compare power of frequency bands between channels.

In the example above you can clearly see that we lose a lot of frequency resolution in the second channel as a result of rescaling the color map from one that is optimized for individual channels to one that is optimized for all channels.
