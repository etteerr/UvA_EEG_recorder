About EEG recorder

 

EEG recorder is a program by Michiel van Wijk (M.vanwijk@uva.nl)

It was created for lab courses at the university of Amsterdam (UvA)

EEG recorder is part of a software set that also includes “Stimulus designer” & “Stimulus presenter”

 

Software requirements

Windows7 64 bit

Matlab 2007 or higher

Data acquisition toolbox (32 bit)

Signal analysis toolbox

Statistics toolbox

G.mobilab+ driver (32 bit and 64 bit) (first install 64 bit, than 32 bit)

G.mobilab+ API fir matlab

 

Hardware requirements

G.mobilab+ (8 channel EEG recorder)
